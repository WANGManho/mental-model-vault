# Workflow Scenarios

## Adding a New Book Idea
1. Capture highlights and bibliography in a fleeting note.
2. Distill the key principle (e.g., Compounding) and store it under `01_Principles`.
3. Create one or more mental models under `02_Mental_Models` describing mechanisms from the book.
4. Link the models to relevant strategies, cases, and biases.

## Preparing for a Decision
1. Browse `07_MOC/Mental Models Map` to select applicable models.
2. Review related cases (`05_Cases`) and known biases (`04_Cognitive_Bias`).
3. Open `Templates/Decision Log Template` to capture context, options, and outcomes.
4. After the decision, set a reminder to revisit the note with actual results.

## Building Cross-Disciplinary Links
- Use tags such as `#economics`, `#psychology`, or `#systems` to cluster notes.
- Add "Related Models" sections when you notice similar structures.
- When two notes share a pattern, create or update a MOC page so the relationship is easy to surface later.
