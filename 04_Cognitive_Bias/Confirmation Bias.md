# Cognitive Bias: Confirmation Bias

## Description
We overweight information that supports existing beliefs and discount contradictory evidence.

## Impact
- Distorts research, causing tunnel vision in strategy work.
- Reinforces echo chambers inside [[Network Effects]] communities.

## Countermeasures
- Seek disconfirming evidence deliberately.
- Pair decisions with red teams or devil's advocates.
