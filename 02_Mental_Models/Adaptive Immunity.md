# Mental Model: Adaptive Immunity

## Definition
Adaptive immunity is the biological system that learns to recognize pathogens and mounts targeted responses that improve with repeated exposure.

## Core Principle
Systems can build memory and pattern recognition, allowing faster, stronger responses to recurring threats.

## When To Use
- Security teams creating playbooks and threat intelligence libraries.
- Customer support knowledge bases that learn from past incidents.

## Examples
- Intrusion detection tuned by previous attack signatures.
- QA test suites expanding after each defect to prevent regressions.

## Related Models
[[Feedback Inhibition]] · [[Homeostasis]] · [[Reference Class Forecasting]]

## Opposite / Failure Cases
- Completely novel threats that bypass existing antibodies or signatures.
- Static checklists that never incorporate past lessons.
