# Mental Model: Sensitivity Analysis

## Definition
Sensitivity analysis tests how changes in inputs affect outputs to identify which assumptions drive results.

## Core Principle
By varying parameters systematically, you learn which levers matter most and where to focus accuracy or hedges.

## When To Use
- Financial models, forecasts, and simulations.
- Product experiments to understand which features influence satisfaction.

## Examples
- Stress-testing a business case by altering churn, price, and cost assumptions.
- Running scenario analysis on supply chain lead times.

## Opposite / Failure Cases
- Purely qualitative decisions with no quantifiable inputs.
- Models with too many interdependencies to vary independently.
