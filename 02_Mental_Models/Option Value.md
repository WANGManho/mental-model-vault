# Mental Model: Option Value

## Definition
Option value captures the benefit of keeping choices open until more information becomes available.

## Core Principle
Flexibility has economic value, especially under uncertainty; reversible decisions can be delayed, while irreversible ones demand extra diligence.

## When To Use
- Staged investments, product experiments, and hiring decisions.
- Negotiations where walk-away power changes leverage.

## Examples
- Running pilot programs before rolling out globally.
- Structuring contracts with extension options instead of long commitments.

[[Superposition]] · [[Margin of Safety]] · [[Regret Minimization]]

## Opposite / Failure Cases
- Opportunities with limited windows where waiting destroys value.
- Commoditized procurement where optionality provides little benefit.
