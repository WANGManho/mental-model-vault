# Mental Model: Frame of Reference

## Definition
A frame of reference is the perspective or coordinate system from which observations are made; different frames can yield different interpretations of the same event.

## Core Principle
Perception, data, and metrics depend on context, so decisions should consider multiple frames to avoid biased conclusions.

## When To Use
- Analyzing metrics that shift with scale or segmentation.
- Resolving stakeholder disagreements stemming from differing vantage points.

## Examples
- Revenue growing overall while a region declines, depending on viewpoint.
- Engineers debating performance using distinct benchmark workloads.

## Opposite / Failure Cases
- Absolute measurements independent of observer (rare outside physics).
- Standardized frameworks forcing everyone into the same context.
