# Mental Model: Diminishing Returns

## Definition
Diminishing returns occur when each additional unit of input yields progressively smaller output gains after a threshold.

## Core Principle
Constraints such as fixed capital, attention, or resources bottleneck productivity, so incremental investment eventually adds less value.

## When To Use
- Staffing projects where coordination costs rise.
- Marketing spend that saturates a channel audience.

## Examples
- Adding engineers to a late project increases communication overhead.
- Fertilizer boosts crop yield until soil chemistry becomes limiting.

## Related Models
[[Bottleneck Analysis]] · [[Opportunity Cost]] · [[Pareto Principle]]

## Opposite / Failure Cases
- Network or flywheel effects where returns accelerate instead of diminish.
- Step-change innovations that reset the production function.

## Sources
- Production theory in economics.
