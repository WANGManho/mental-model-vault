# Mental Model: Reference Class Forecasting

## Definition
Reference class forecasting improves predictions by comparing the current project to a class of similar past projects rather than relying solely on inside views.

## Core Principle
Base rates anchor expectations, counteracting optimism bias and ensuring estimates reflect historical distributions.

## When To Use
- Large capital projects, software initiatives, or hiring plans.
- Any forecast where teams tend to underestimate cost or time.

## Examples
- Estimating implementation timelines based on actual durations of comparable rollouts.
- Budgeting new products using gross margin trajectories from earlier launches.

## Opposite / Failure Cases
- Truly novel projects with no meaningful reference class.
- Environments undergoing structural break where history misleads.
